# VIDZ PLAY LITE — Project Evolution

VIDZ PLAY LITE is the lightweight, browser-based version of VIDZ PLAY. This document describes how the project evolved from a simple local video player into a complete live audiovisual instrument.

## Design Principles

- One HTML file
- Local-first playback
- Fullscreen performance
- Hardware-inspired interface
- Immediate interaction

## V1 — Local Video Player

The first version loaded local videos and automatically played them fullscreen. The objective was to transform a browser into a simple performance instrument.

## Folder Import

Support for complete video folders turned the application into a playable visual library instead of a simple file player.

## Mobile Version

The interface was redesigned for touch interaction, portrait and landscape layouts, wake lock support and fullscreen operation on phones and tablets.

## Samsung Galaxy Z Flip6 Edition

A dedicated interface was designed for the Samsung Galaxy Z Flip6. The phone becomes a portable VJ controller able to browse local video and audio folders, launch a mix with a single LET'S GO button and drive an external HDMI display in either 16:9 crop or 9:16 vertical mode.

## Quad Buffer Engine

Playback evolved from a single video element to a four-deck architecture. Hidden players continuously prepare upcoming clips, greatly reducing visible loading gaps.

## SCAN → FILTR → PLAY

VIDZ SCAN analyses media, VIDZ FILTR organises collections and VIDZ PLAY LITE performs them in real time.

## Visual Engine

The graphics engine progressively added hue, contrast, pulse, flash, strobe, RGB split, zoom, VHS, glitch and frame-jump effects.

## Beat Reactive Engine

Web Audio analysis introduced beat-driven behaviour, allowing visual events to react to bass energy and musical transients.

## Beat Scratch

The scratch engine performs a very short backward movement of the playback head on selected beats before immediately resuming normal playback. The image never intentionally freezes and the effect is designed to feel like manipulating a physical playback device.

## Recording

REC evolved into a canvas-based recorder capturing the final rendered composition, including transitions, layouts and effects.

## Current Features

- Local folder playback
- Audio import
- Quad buffer engine
- Double-screen mode
- Portrait and landscape support
- Mobile interface
- Samsung Galaxy Z Flip6 edition
- SCAN / FILTR compatibility
- Beat detection
- Beat scratch
- VHS and glitch engine
- Final-output recording
